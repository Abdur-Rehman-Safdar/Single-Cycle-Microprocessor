# Single-Cycle-Microprocessor
A complete implementation of a 32-bit Single-Cycle RISC-V Processor based on the RV32I instruction set architecture, developed in Verilog HDL for learning processor architecture and digital design.

Features
✅ Single-Cycle RISC-V Datapath
✅ Verilog HDL Implementation
✅ Modular RTL Design
✅ Main Decoder & ALU Decoder
✅ 32×32 Register File
✅ Instruction & Data Memory
✅ Immediate Extension Unit
✅ ALU (ADD, SUB, AND, OR)
✅ Branch and Jump Control Logic
✅ JAL (Jump and Link) Support
✅ Functional Simulation & Waveform Verification
✅ Self-Checking Testbench
Supported Instructions: ADD, SUB, LW, SW, BEQ, JAL
Tools Used: Verilog HDL, Xilinx Vivado Simulator, RTL Design & Functional Verification
Learning Outcomes: Processor Datapath Design, RTL Development, Control Unit Design, Computer Architecture,
RISC-V ISA, Digital Logic Design, Functional Verification, Hardware Debugging
